There are a ton of job boards. Keeping track of new jobs is hard. With Local Remote Jobs Aggregator you have a centralized place of all remote jobs scrapped by the provided chrome extension. You can extend it with other job boards just by adding another function in the chrome extension source code.
Start the server, start the chrome extension, wait for chrome extension to do the scrapping, open localhost:3000 (or other port if that was changed) and start job hunting.
