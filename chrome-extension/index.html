Local Remote Dev. Jobs Agg.

Start the server to receive incoming collected jobs then click 'Collect jobs' button. After it's done, keep the server running and click 'View jobs' button. You can use Sqlite DB Browser to filter them in bulk before accessing the ui. Click 'Reset storage' button in case 'Collect jobs' failed or job websites close on load. If some pages remain open accept cookies/modals and refresh the page. While looking at job description disable "job collection" and enable "job description scanning".
